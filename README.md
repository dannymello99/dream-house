# 💛 Dream House — Vaquinha Solidária

Este projeto é um site interativo de vaquinha online criado com o objetivo
de ajudar na realização do sonho de morar sozinho e iniciar uma nova fase da vida.

---

## 🌐 Acesse o site

👉 https://vaquinhaOnline.github.io/dream-house

---

## ✨ Funcionalidades

✅ Página moderna e interativa  
✅ Fundo visual estilizado  
✅ Botão direto para contribuição via Pix  
✅ Copiar chave Pix automaticamente  
✅ Contador de progresso da meta  
✅ Página de agradecimento animada  
✅ Contato rápido via WhatsApp  

---

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages (Hospedagem gratuita)

---

## 🎯 Objetivo do projeto

Criar uma página simples, acessível e confiável para arrecadação online,
proporcionando uma experiência agradável para quem deseja ajudar.
